## AuctionDrafter  

### Description
Auction Style 5vs5 Team Drafter GUI made in python3(pyqt5)  
2 Team Leaders can bid their resources to pick 4 teammates, it makes 2 teams of 5

### USAGE  
python3 gui.py

### Environment  
Python 3.7.4  
pyqt5
